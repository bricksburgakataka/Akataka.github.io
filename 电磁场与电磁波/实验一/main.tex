\section*{三、思考题解答}
\subsection*{1. 简述 S 参数的定义及物理意义，说明 S11、S21 的含义}

\textbf{S参数（散射参数）}：描述微波网络中入射波和反射波之间关系的参数，用于表征网络的传输和反射特性，适用于高频信号分析。

\begin{itemize}
    \item \textbf{S\(_{11}\)}（输入反射系数）：端口2匹配时，端口1的反射波与入射波之比。物理意义为端口的\textbf{回波损耗}，反映端口匹配程度，S\(_{11}\)越小（越负）表示匹配越好。
    \item \textbf{S\(_{21}\)}（正向传输系数）：端口2匹配时，端口2输出波与端口1输入波之比。物理意义为\textbf{插入损耗}或\textbf{增益}，反映信号从端口1传输到端口2的能力。
\end{itemize}

\subsection*{2. 若不进行校准，直接接入射频电缆和电路模块测量，会对结果产生什么影响？请分析原因}

\textbf{影响}：
\begin{itemize}
    \item 测量结果包含系统误差（电缆损耗、转接头反射、端口失配等），无法得到DUT的真实参数
    \item S\(_{11}\) 读数偏大（匹配变差），S\(_{21}\) 读数偏小（损耗虚高）
    \item 相位测量不准确，影响阻抗、时域分析等
\end{itemize}

\textbf{原因}：
\begin{itemize}
    \item 矢量网络分析仪通过校准消除系统误差项（方向性、源匹配、负载匹配、隔离、跟踪等）
    \item 不校准时，电缆的损耗、驻波、长度延时等会叠加到DUT特性中，导致测量结果失真
\end{itemize}

\subsection*{3. 设计实验方案，测量 N-SMA 转接头对测试曲线的影响}

\textbf{实验方案}：
\begin{enumerate}
    \item \textbf{校准}：使用N型校准件对VNA进行双端口SOLT校准，校准面设在N型端口。
    \item \textbf{直通测量}：两根N型电缆直接相连，测量S\(_{11}\)、S\(_{21}\)作为参考基线。
    \item \textbf{接入转接头}：在一条通路中串入N-SMA转接头 + SMA电缆 + SMA-N转接头。
    \item \textbf{对比分析}：
        \begin{itemize}
            \item 对比两次测量的S\(_{11}\)（反射）变化，评估转接头引入的驻波
            \item 对比S\(_{21}\)幅度和相位变化，评估插入损耗和相位偏移
        \end{itemize}
    \item \textbf{时域分析}（可选）：通过VNA的低通时域模式，观察转接头位置产生的反射脉冲。
\end{enumerate}

\subsection*{4. 根据实验给出的微带线设计参数，计算 50 Ω 半波长微带线的宽度和长度}

\subsubsection*{已知条件}
\begin{itemize}
    \item 工作频率：\( f_0 = 2.5 \, \text{GHz} \)
    \item 特性阻抗：\( Z_0 = 50 \, \Omega \)
    \item 相对介电常数：\( \varepsilon_r = 4.6 \)
    \item 介质厚度：\( h = 0.765 \, \text{mm} \)
\end{itemize}

\subsubsection*{宽度 \( W \) 计算}

中间量 \( A \)：
\[
A = \frac{Z_0}{60}\sqrt{\frac{\varepsilon_r+1}{2}} + \frac{\varepsilon_r-1}{\varepsilon_r+1}\left(0.23+\frac{0.11}{\varepsilon_r}\right) \approx 1.558
\]

\[
\frac{W}{h} = \frac{8e^A}{e^{2A}-2} \approx 1.85
\]

\[
W = 1.85 \times 0.765 \approx \boxed{1.41 \, \text{mm}}
\]

\subsubsection*{有效介电常数 \( \varepsilon_{re} \) 计算}

\[
\varepsilon_{re} = \frac{\varepsilon_r+1}{2} + \frac{\varepsilon_r-1}{2} \cdot \frac{1}{\sqrt{1+12h/W}} \approx \boxed{3.458}
\]

\subsubsection*{半波长长度 \( L \) 计算}

真空中波长：
\[
\lambda_0 = \frac{c}{f_0} = \frac{3\times10^8}{2.5\times10^9} = 120 \, \text{mm}
\]

导波波长：
\[
\lambda_g = \frac{\lambda_0}{\sqrt{\varepsilon_{re}}} \approx \frac{120}{\sqrt{3.458}} \approx 64.53 \, \text{mm}
\]

半波长长度：
\[
L = \frac{\lambda_g}{2} \approx \boxed{32.26 \, \text{mm}}
\]

\begin{itemize}
    \item 实验中滤波器插入损耗偏大，分析为FR4基板损耗高和端口匹配不佳所致，后续可改用低损耗基板或优化匹配结构
    \item 天线驻波比在2.45 GHz以上恶化，说明设计带宽不足，可通过调整辐射贴片尺寸或增加匹配网络来改善
\end{itemize}
